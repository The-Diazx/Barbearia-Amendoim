# 💈 Barbearia Amendoim

> Um site institucional e de agendamento moderno para a Barbearia Amendoim, desenvolvido para focar na experiência do usuário e na facilidade de acesso aos serviços.

---

## 💻 Sobre o Projeto

O **Barbearia Amendoim** é uma aplicação web desenvolvida para apresentar os serviços, profissionais, horários de atendimento e a história da barbearia. O objetivo principal é fornecer aos clientes uma interface limpa, atraente e responsiva para conhecer o estabelecimento e realizar agendamentos.

### ✨ Funcionalidades

- **Página Inicial:** Apresentação da barbearia com design moderno e chamadas para ação (CTA).
- **Catálogo de Serviços:** Listagem detalhada de serviços (corte de cabelo, barba, tratamentos) com respectivos preços.
- **Sobre Nós:** Seção dedicada à história da barbearia e à missão da equipe.
- **Contato & Localização:** Integração com mapas, endereço, links para redes sociais e botão direto para o WhatsApp.
- **Responsividade:** Interface totalmente adaptável para dispositivos móveis, tablets e desktops.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização, layout (Flexbox/Grid), animações e responsividade.
- **JavaScript:** Interatividade, manipulação do DOM e possíveis integrações de agendamento.
